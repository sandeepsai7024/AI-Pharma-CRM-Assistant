# AI Pharma CRM Commercial Assistant

[![Live Demo](https://img.shields.io/badge/Live-Demo-emerald?style=for-the-badge&logo=google-cloud)](https://ais-pre-btbmsoopkl4saztinqojly-85305307370.europe-west1.run.app)

An enterprise-grade AI-powered pharmaceutical engagement dashboard built with **React**, **Tailwind CSS**, and **Google Gemini AI**. This tool helps pharmaceutical representatives prepare personalized, compliant doctor engagement strategies.

## 🔗 Live Application
**Link:** [https://ais-pre-btbmsoopkl4saztinqojly-85305307370.europe-west1.run.app](https://ais-pre-btbmsoopkl4saztinqojly-85305307370.europe-west1.run.app)

## 🚀 Features

- **Personalized Doctor Engagement Summary**: AI-generated profiles based on HCP history.
- **Compliant Follow-Up Suggestions**: Ensures all communication adheres to pharmaceutical ethical standards.
- **Next Best Action (NBA) Engine**: Predictive recommendations for the next engagement step.
- **Omnichannel Strategy**: Integrated ideas for Email, WhatsApp, Webinar, and In-person meetings.
- **Risk Analysis**: AI-driven alerts for low engagement or specific patient adherence concerns.
- **Professional Dashboard UI**: Clean, responsive, and healthcare-oriented interface.

## 🛠️ Tech Stack

- **Frontend**: React 19 + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **AI**: Google Gemini API (gemini-2.0-flash)
- **Icons**: Lucide React

## 📦 Installation

1. **Clone the repository** (if applicable).
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment Variables**:
   Create a `.env` file (or use the one provided in AI Studio) and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_key_here
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```

## 📈 Pharma Business Relevance

In the highly regulated pharma industry, personalization is key to successful medical engagement. This assistant:
1. **Reduces Prep Time**: Automates the analysis of CRM history.
2. **Enhances Compliance**: Safeguards against unapproved or exaggerated claims.
3. **Improves HCP Relationships**: Delivers high-value clinical data that doctors actually need.
4. **Data-Driven Growth**: Uses "Risk Alerts" to identify accounts that need urgent attention.
